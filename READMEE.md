# Тема "Портфолио"

Это кастомная тема WordPress, разработанная для создания и управления сайтом-портфолио.

## Функционал

### Тип контента "Портфолио"

Тема регистрирует специальный тип контента "Портфолио" (`portfolio`), который полностью отделен от стандартных записей блога. Это позволяет удобно управлять работами, не смешивая их с новостными или информационными статьями.

### Продвинутая система управления ролями

Для гибкого распределения прав доступа к контенту в теме реализована продуманная система ролей.

#### Кастомная роль: "Менеджер Портфолио"

Для делегирования управления портфолио создана специальная роль "Менеджер портфолио". Она идеально подходит для сотрудника или фрилансера, который должен заниматься только наполнением раздела с работами.

**Возможности роли "Менеджер портфолио":**
*   Создавать, редактировать и публиковать свои собственные работы в разделе "Портфолио".
*   Редактировать чужие работы в портфолио (например, созданные администратором).
*   Загружать медиафайлы для своих работ.

**Ограничения роли "Менеджер портфолио":**
*   Не видит и не может управлять разделом "Записи" (Блог).
*   **Не может удалять чужие работы в портфолио.**
*   Не имеет доступа к управлению страницами, пользователями, настройками сайта, плагинами или темами.

#### Автоматическое управление ролью

Роль "Менеджер портфолио" не засоряет систему WordPress. Она **автоматически создается** при активации темы "Портфолио Креатора" и **автоматически удаляется** при переключении на любую другую тему.

#### Использование стандартных ролей

*   **Автор (Author)**
    Используется для гостевых блогеров. Может создавать, редактировать и публиковать только свои записи в Блоге. Не имеет доступа к разделу "Портфолио".

*   **Подписчик (Subscriber)**
    Используется для клиентов. Может войти на сайт (например, для просмотра скрытого контента) и редактировать свой профиль, но не имеет доступа к консоли управления сайтом.
